# Chat-application

## Features 
- One to One chat option
- One-to-Many Chat option
- Ability to create multiple Groups
- Users can join Available Groups
- User can Toggle Themes
- Authentication / Authorization
- Chat Backup to database

## App Structure
- User Interface
- Server
- Database
